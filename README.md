## neovim
1. `sudo apt insall git`
2. `sudo apt-get install manpages-dev`
3. `sudo add-apt-repository ppa:neovim-ppa/unstable`
4. `sudo apt-get update`
5. `sudo apt-get install neovim`
6. Make sure you don't already have `~/.config/nvim` directory. If you have remove it.
7. Clone [this](https://github.com/LunarVim/Neovim-from-scratch) repo into `~/.config/nvim`
